make_missionoptions_python.py¶

This file contains the make_PyEMTG_MissionOptions function.

make_missionoptions_python.make_PyEMTG_MissionOptions(OptionsDefinitions, now, path='.')[source]¶

Function for automatically creating a MissionOptions.py file based on a csv of mission options.

Parameters:

  • OptionsDefinitions (List of dict entries) – Dict entries that define the options to be added to MissionOptions.py

  • now (string) – A string of the current time.

  • path (string, optional) – Path to directory in which to write MissionOptions.py. The default is ‘.’

Return type:

None.
